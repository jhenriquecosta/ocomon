Centros de Custo (Admin -> Centros de Custo)

Centro de custo é um identificador contábil do departamento proprietário do equipamento.

O OcoMon permite que se cadastre os Centros de custos em sua própria base ou acesse essas informações de alguma outra base desde que esta seja em MySQL. Para isso, é necessário configurar o arquivo de configurações do OcoMon (ocomon/includes/config.inc.php).
