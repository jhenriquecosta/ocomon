Fornecedores (Admin -> Inventário -> Fornecedores)

A partir dessa tela é possível visualizar e cadastrar fornecedores de equipamentos e softwares.

Fornecedores, dentro do OcoMon, são empresas que vendem soluções de hardware e software. Essa informação é útil para contato em caso de acionamento de garantia para os equipamentos ou atualização de software por exemplo.
