Meus Chamados (Home -> Meus Chamados)

Nessa área é possível ter acesso rápido aos chamados que o operador logado participou. As ocorrências são exibidas em uma tela dividida em 3 painéis paginados onde:

Primeiro painel: exibe a listagem das ocorrências abertas pelo operador logado e ainda ativas(não concluídas) no sistema.

Segundo painel: exibe a listagem de todas as ocorrências abertas pelo operador logado e já concluídas no sistema.

Terceiro painel: exibe a listagem de todas as ocorrências concluídas pelo operador logado no sistema. Para usuários de nível somente-abertura esse painel não traz informações, visto que esse tipo de usuário não encerra chamados no sistema.
