Esquema de Cores (Home -> Esquema de Cores)

O OcoMon permite a utilização de diferentes esquemas de cores em seu layout. Cada operador pode selecionar um dos temas disponíveis para personalizar o ambiente do OcoMon.

Atenção: a seleção só estará disponível se for habilitada pelo administrador em Admin -> Configurações -> Abertura de chamados.
