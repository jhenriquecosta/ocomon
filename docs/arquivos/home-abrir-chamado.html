Abrir Chamado (Home -> Abrir Chamado)

Essa opção só visível para usuários de nível somente-abertura.

Ao clicar nessa opção, será exibido o formulário para abertura de chamados. Esse formulário pode ser customizado em Admin -> Configurações -> Abertura de chamados, na versão 2.0rc3, e em Admin -> Configurações -> Perfis de tela de abertura, a partir da versão 2.0RC4.
