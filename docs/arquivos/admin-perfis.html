Menu Admin -> Configurações -> Perfis de tela de abertura

A partir da versão 2.0RC4 é possível criar diversos perfis de telas de abertura de chamado. Um perfil de tela de abertura nada mais é do que um conjunto de definições sobre quais campos serão visíveis na tela de abertura e para qual área de atendimento serão direcionados os chamados abertos por seus usuários quando o campo "Área de atendimento" não fizer parte dos campos visíveis do perfil relacionado. É a customização da tela de abertura de chamados em função de cada área de atendimento.

Para a criação e edição de um perfil basta acessar o menu Admin->Configurações->Perfis de tela de abertura
Na tela citada não é possível associar as áreas de atendimento a cada perfil, para isso, é necessário acessar o menu de administração de áreas de atendimento em: Admin->Ocorrências->Áreas

Um perfil pode ser associado a diversas áreas de atendimento. Áreas de atendimento que não forem associadas à um perfil específico terão a tela de abertura completa, com todos os campos disponíveis.
