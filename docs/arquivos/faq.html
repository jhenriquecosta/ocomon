Perguntas Frequentes

Nessa área foram reunidas as principais perguntas (e respostas) de usuários novatos no sistema.

Perguntas:

01 - Não consigo me logar no sistema com o usuário "admin" e senha "admin".

02 - Após logar no sistema o mesmo entra em loop e fica dando refresh constantemente.

03 - O sistema não está exibindo corretamente as palavras com acentuação.

04 - Primeiro acesso, o sistema exibe um monte de texto puro na tela e não mostra o layout correto.



Respostas:

01 - Até a versão 1.40 o OcoMon precisa que a diretiva register_globals esteja marcada como On. Essa configuração deve ser feita no arquivo de configuração do PHP (php.ini). Dependendo do sistema é comum existirem mais de um arquivo php.ini, portanto, antes de editar o arquivo tenha certeza de qual php.ini está sendo utilizado pelo PHP.


02 - Essa situação ocorre caso você esteja acessando o sistema com usuário de nível somente-abertura mas associado a uma área que esteja marcada como "Atende chamados".


03 - Edite o arquivo de configuração do apache (httpd.conf ou apache2.conf). defina como default a codificação iso-8859-1 no charset. para isso, verifica a entrada "adddefaultcharset", se estiver utf-8, você comenta e adiciona a linha: adddefaultcharset iso-8859-1. Caso não exista a entrada "adddefaultcharset" apenas adicione a linha adddefaultcharset iso-8859-1 no arquivo.
Após isso é necessário reiniciar o apache.


04 - Até a versão 2.0rc3 é necessário habilitar a diretiva short_open_tag no arquivo de configuração do PHP (php.ini) para que esse problema não ocorra.
